# Arquitectura Inicial SIAB

El proyecto usa un paquete central `config` y seis apps Django dentro de `apps`.

Apps:

- configuracion
- socios
- jugadores
- finanzas
- bingos
- seguridad

Entidades existentes para mapear posteriormente:

- TipoSocio
- Socio
- CuentaBancaria
- MetodoPago
- Prestamo
- Pago
- Bingo
- Ahorro
- PartidaBingo
- Jugador
- Carton
- PlataformaJuego
- SesionJuego
- Regalo
- AporteSemanal

Los modelos se deben declarar respetando exactamente nombres de tablas, columnas, claves primarias y claves foraneas aprobadas.
